NG Store

Main Features:

Authentication

The user must be logged in in order to navigate threw the app. You can use elikopter/P@ssw0rd credentials to do so.

Orders History

First, choose a customer from the customers dropdown list. Then an other dropdown appears with the order list of the chosen customer.
Select an order, and a table pops up with the order's details.

New Order

There is a form to fill in.
As before, choose a customer from the dropdown list. Then choose a product and its quantity which must be at least one.

An add button appears. Click it if you want to add an other product to the list.
Notice that the same product cannot be chosen twice or more.
If there are at least two products in the order list, you can remove each of them by clicking on the trash button.

Finally, there is a checkbox that set on true will give the customer 10% discount.

Once the form is valid, the user can submit it after having confirmed.
A pop up message appears (success/error) accordingly to the server's response.

Finally, the user is redirected to the "orders" page.
